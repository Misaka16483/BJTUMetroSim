# 9号线供电拓扑V1参数与溯源

## 1. 文档目的

本文档说明 `data/scenarios/line9_power_topology.json` 中教学版供电拓扑的证据、假设和替换边界。该配置用于DC750V准静态牵引供电研究，不作为施工图、继电保护整定或运营倒闸依据。

配置版本为 `LINE9-DC750-V1.0`，质量等级为 `ENGINEERING_ESTIMATE`。系统必须将这一等级传递到API、记录和页面，禁止把工程估算参数标记成实测数据。

## 2. 已确认事实与工程假设

| 内容 | V1取值 | 证据等级 | 说明 |
|---|---:|---|---|
| 牵引供电制式 | DC750V接触轨、走行轨回流 | 运营方公开信息 | 北京地铁公开供电知识支持该制式和单双边供电方式 |
| 牵引变电所数量 | 10座 | 公开工程资料 | 早期9号线环评资料支持10座地下牵引变电所 |
| 郭公庄牵引所 | `TS-0901` | 公开信息锚点 | 郭公庄存在牵引混合变电站的公开证据较强 |
| 其余牵引所位置 | 按线路公里标和供电半径布置 | 工程估算 | 未获得竣工一次系统图，位置可被后续实测/图纸替换 |
| 正线供电方式 | 相邻所小双边供电 | 工程规则 | 中间所左右、上下行分别设置馈电臂 |
| N-1越区方式 | 故障所馈线断开、边界联络开关闭合 | 工程规则 | 表示大双边供电的教学抽象 |

## 3. 参数基线

| 参数 | V1值 | 来源ID | 质量与用途 |
|---|---:|---|---|
| 标称电压 | 750 V | `SRC-BJ-SUBWAY-DC750` | 公开制式参数 |
| 整流侧空载电压 | 825 V | `SRC-DC750-EQUIVALENT-V0` | 论文与工程区间中选取的等值参数 |
| 牵引所等效内阻 | 0.015 ohm | `SRC-DC750-EQUIVALENT-V0` | 0.010至0.020 ohm区间的V1中值 |
| 牵引所额定/过载电流 | 5300/8000 A | `SRC-DC750-CAPACITY-V0` | 教学容量与保护阈值，待设备资料校准 |
| 馈线电缆等效电阻 | 0.0036 ohm | `SRC-POWER-RESEARCH-V0` | 2根500 mm2铜缆、约0.2 km的估算值 |
| 接触轨纵向电阻 | 0.0083 ohm/km | `SRC-POWER-RESEARCH-V0` | 文献参数区间内的教学取值 |
| 回流轨纵向电阻 | 0.0083 ohm/km | `SRC-POWER-RESEARCH-V0` | 走行轨回流等效取值 |
| 端部能馈容量 | 1000 kW/所 | `ASSUMPTION-EFS-ENDS-V0` | 仅用于再生回馈研究，不代表9号线实际配置 |

## 4. 显式拓扑规则

1. 10座牵引所按公里标严格递增。
2. 两个端部牵引所各设置2条正线馈电臂；8个中间所各设置4条，共36条。
3. 相邻牵引所之间按上下行分别设置接触轨段和回流段，各18段。
4. 每个内部边界设置1个常开联络开关，共9个。
5. 设备必须包含 `sourceId`、`quality` 和 `parameterSources`；缺少任一项时严格加载失败。
6. 正式配置不得依赖 `_generate_v0_sections` 静默补齐。生成函数仅保留给显式声明 `allowGeneratedTopology=true` 的临时迁移数据。

## 5. 校准与替换顺序

后续获得一次系统图、PSCADA点表或设备参数时，按以下顺序替换：牵引所准确位置和类型、绝缘分段与馈线范围、整流机组数量与容量、馈线电缆参数、接触轨和回流轨电阻、联络开关逻辑、能馈装置位置与能力。

每次替换必须提升对应设备的 `quality`，保留原始来源ID，并重新执行双边供电解析算例、N-1、能量守恒和长时仿真验收。
