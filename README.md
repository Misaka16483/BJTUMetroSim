# BJTUMetroSim

北京交通大学软件工程实训 — 北京地铁仿真系统

## 快速启动

```bash
cd bj-metro-sim
npm install
npm run dev
```

浏览器访问 `http://localhost:5173/`

### 技术栈

- React 19 + TypeScript
- Vite 6
- MapLibre GL JS（MapTiler 暗色矢量地图）
- Ant Design 5 + Tailwind CSS 4
- Zustand 5（状态管理）
- OpenStreetMap Overpass API（北京地铁真实线路数据）
